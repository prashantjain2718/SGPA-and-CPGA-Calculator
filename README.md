## ğŸ“– How It Works

### 1. The Formula
The calculator uses the official RTU formulas:

* **SGPA** = $\frac{\sum(Credit \times GradePoint)}{\sum(TotalCredits)}$
* **CGPA** = $\frac{(PrevCGPA \times PrevCredits) + (CurrSGPA \times CurrCredits)}{TotalCredits}$

### 2. Grading Scale
| Grade | Points | Description |
| :--- | :---: | :--- |
| **A++** | 10 | Outstanding |
| **A+** | 9 | Excellent |
| **A** | 8.5 | Very Good |
| **B+** | 8 | Good |
| **B** | 7.5 | Above Average |
| ... | ... | ... |
| **F** | 0 | Fail |
